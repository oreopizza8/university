## manual_supplement_jeongsi.csv 상세 (입학처 수동수집 보충)
- **목적:** 어디가 지역 PDF가 이미지/백분위 미공개로 못 담은 대학을, 대학 입학처 공시 입시결과로 보충. 자동추출(`integrated_*`)과 **분리 보관**해 재추출 시 보존됨. `build_app_data.py`가 `extra=` 로 병합.
- **현재 수록: 한양대학교 54개 학과** (출처: 한양대 입학처 '2024~2026 주요 전형 입시결과', 2026학년도 정시 일반).
  - ⚠️ **단위 주의:** 한양대 공시값은 "최종등록자 **상위 80% 평균** 국/수/탐 백분위"이며, 우리 시스템의 "70%컷"과 **정의가 다름**. 스크리닝 참고용으로는 동급 백분위라 사용 가능하나, 정밀 비교 시 메타 차이 인지 필요. (예: 한양대 경영 92.12, 의예 98.38)
  - 반영비율은 계열 추정치(타 대학과 동일 기준).
- **수집 불가로 제외(2026-05-21 확인):** 중앙대(경쟁률·추합만, 점수컷 미공시), 한국외대(환산점수 자체척도), 이화여대·홍익대(모집요강만, 입결 컷 없음). → 이들은 백분위/우리척도로 표현 불가 = 컨설팅 영역.

## kcue_domestic_cutoffs.csv 상세
- **추출:** `convert_ipgyeol_pdf_to_csv.py` — 간지 큰 글씨로 대학명 인식 + 정시 표 "환산점수 만점(800/1000) 백분위" 패턴으로 70%컷 백분위.
- **반영비율:** PDF에 없어 계열(자연/인문/기타)별 표준값 적용 = **추정치**. 정확값은 대학 모집요강 기준 보강 필요.
- **범위 한계:** 정시 백분위를 공개한 대학만 수록. 백분위 미공개(환산점수/등급만) 대학 다수는 제외. 서울대·연세대·서강대·한양대 등은 제공 PDF(`_1`)에 없어 시드(20행)로만 일부 커버 → 서울_2,_3 또는 어디가 정시결과 추가 필요.
- **갱신:** 새 PDF를 이 폴더에 넣고 `python convert_ipgyeol_pdf_to_csv.py *.pdf` → `POST /api/ingest/staging/reload`.
