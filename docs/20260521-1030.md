- **문제:** 대학별 중앙값±20pp(정시)/±2.5등급(수시) 필터가 **지방 의대·한의대·약대(최상위)·예체능(편차 큼)**을 추출오류로 오인해 삭제(연세 예체능, 동의대 한의 93.7, 강원대 의예 97.2 등).

### B. 정시 풍부 스키마 재추출 (`extract_adiga_2025.py` 전면 확장)
- 기존 검증된 percentile 로직(Format A 단일평균 / Format B 과목별 평균, 연세 93.8) **보존**하면서 컬럼 추가.
- **PDF 표 구조 확정:** Format B(연세 등 25컬럼) = `0구분 1모집단위 2최초 3이월 4최종(A+B) 5경쟁률 6충원합격순위 7환산50 8환산70 9총점 | 16국 17수 18탐1 19탐2 20영(등급) 21한(등급) | 22~24응시비율`. Format A(서울교대 등) = 평균 백분위 단일칸(과목컷 없음).
- **신규 CSV 스키마** `integrated_jeongsi_results_2025.csv`(utf-8-sig):
  `region, universityName, admissionGroup, departmentName, recruitCount, competitionRate, additionalRank, koreanCut70, mathCut70, tamguCut70, percentile70(=수능평균70%컷·범위검색 메인인덱스), ratioKor, ratioMath, ratioEng, ratioTam`
  - `percentile70`을 그대로 유지 → 하위 `build_app_data.py`/`StagingDataLoader` 호환.
  - 경쟁률 `6.8:1`→6.8 정규화, 추합 `-`→공백, ratio는 계열 추정치(README 신뢰선).
- **결과:** 정시 **1,682행/62대학**, 수시 2,395행/51대학. 검증: 연세 경영 93.8(국98/수90/탐93.5·모집166·경쟁3.58·추합119), 강원 의예 97.2, 고려 경영대학 94.0(Format A 과목컷 공백 정상).
- 정제 후 적재본: `kcue_domestic_cutoffs.csv` 1,584행 / `kcue_susi_cutoffs.csv` 2,332행.
